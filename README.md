## About the Web-Based Corn Maturity Predictor

The Web-Based Corn Maturity Predictor built using Laravel Web Application Framework and OpenCV-Python with CRUD Basic Operation.

- Image Processing
- Color Detection Algorithm
- Data Analytics 
- Maturity Prediction

## Built Using

- Laravel Web Framework 8.28.0
- JQuery AJAX
- OpenCV-Python
- SweetAlert2
- LAMP (Software Bundle)

## Functionality

- Create New Batch
- Upload New Corn Image
- Update and Delete Table Entry
- Search and Sort Datatables
- Filter by Batch Number
- View Table Results
